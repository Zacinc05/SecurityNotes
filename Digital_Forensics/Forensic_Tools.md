Write Blockers

- Block any evidence alteration actions  
- Ensure data integrity during the collection

Autopsy [https://www.autopsy.com/](https://www.autopsy.com/)

- Open-source digital forensics platform  
- Investigator can import an acquired disk image into this tool, and the tool will conduct an extensive analysis of the image  
- Offers various features during image analysis, including keyword search, deleted file recovery, file metadata, extension mismatch detection, etc.

DumpIt [https://www.toolwar.com/2014/01/dumpit-memory-dump-tools.html](https://www.toolwar.com/2014/01/dumpit-memory-dump-tools.html)

- Offers the utility of taking a memory image from a Windows operating system  
- Creates memory images using a command-line interface and a few commands

Volatility [https://volatilityfoundation.org/](https://volatilityfoundation.org/)

- Open-source tool for analyzing memory images
