pdfinfo

- Displays various metadata related to a PDF file, such as title, subject, author, creator, and creation date  
- ‘sudo apt install poppler-utils’

exiftool

- Read and write metadata in various file types, such as JPEG images  
- ‘sudo apt install libimage-exiftool-perl’  
- Filter for information (i.e. | grep “GPS”, | grep “Model”)
