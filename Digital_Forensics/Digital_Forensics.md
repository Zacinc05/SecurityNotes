Computer Forensics

- Computer forensics  
  - Computers/devices  
- Mobile forensics  
  - Mobile devices and extracting evidence (i.e. calls/texts)  
- Network forensics  
  - The whole network (i.e. network traffic logs)  
- Database forensics  
  - Intrusion into these databases that results in data modification or exfiltration  
- Cloud forensics  
  - Investigating data stored on cloud infrastructure  
- Email forensics  
  - Emails are investigated to determine whether they are part of phishing or fraudulent campaigns

Chain of Custody

- Documenting the complete journey of evidence during a legal case lifetime, from the collection to final presentation in court

Window’s Forensics

- As part of the data collection phase, forensic images of Windows operating system taken  
  - Forensic images are bit-by-bit copies of the whole operating system  
- FTK Imager  
  - Tool for taking disk images of Windows operating systems  
- Two categories of forensic images are taken from Windows OS  
  - Disk image  
    - Data present on the storage device of the system (HDD, SSD, etc.)  
    - Non-volatile  
  - Memory image  
    - Memory image contains the data inside the operating system’s RAM  
    - Volatile
