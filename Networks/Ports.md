Ports and Services

- Connection  
  - Physical docking point for connecting external hardware  
  - Virtualized, numbered endpoint for network communication  
- 65,535 available ports  
- Port number tells the computer which program (service) should receive that data  
- Well-Known Ports (0-1023)  
  - Reserved for standard system services  
- Registered Ports (1024-49151)  
  - Used by other applications/services  
- Dynamic/Private Ports (49152-65535)  
  - Used temporarily by your computer when you browse the web.

Router

- Device that allows port forwarding

Port Forwarding

- Allow external internet traffic to access specific devices/services in a private network  
- AKA allowing entry from what was an intranet  
- Intranet  
- Secure/Private/Internal network for an organization
