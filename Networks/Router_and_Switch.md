Router

- Connect networks and pass data between them via routing  
- Routing  
  - Process of data travelling across networks  
  - Useful when devices are connected by many paths  
- Dedicated devices and do not perform the same functions as switches  
  - What path is the shortest  
  - What path is the most reliable  
  - Which path has the faster medium (e.g. copper or fibre)  
- Layer 3 on OSI model

Routing Protocols

- OSPF (Open Shortest Path First)  
- EIGRP (Enhanced Interior Gateway Routing Protocol)  
- BGP (Border Gateway Protocol)  
- RIP (Routing Information Protocol)

Switch

- Dedicated networking device responsible for providing a way to connect to multiple devices  
- Facilitate many devices with ethernet cables  
- Layer 2 & 3 of OSI model
