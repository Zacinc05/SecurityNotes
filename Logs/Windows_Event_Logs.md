Event Viewer

- Windows OS utility  
- Interface to view/search for anything in logs  
- Use  
  - Click ‘Windows Logs’ to view the types of logs  
  - Double click on any log type to open specific info  
  - Actions bar on right for settings/filter  
- Event fields  
  - Description  
  - Log Name  
  - Logged (time)  
  - Event ID (activity type/identifier)

Event ID examples

- 4624	A user account successfully logged in  
- 4625	A user account failed to login  
- 4634	A user account successfully logged off  
- 4720	A user account was created  
- 4724	An attempt was made to reset an account’s password  
- 4722	A user account was enabled  
- 4725	A user account was disabled  
- 4726	A user account was deleted
