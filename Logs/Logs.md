Log

- Digital footprints left behind from activity

Log Use Cases

- Security Events Monitoring  
- Incident Investigation and Forensics  
- Troubleshooting  
- Performance Monitoring  
- Auditing and Compliance

Log Types

- System Logs  
  - Operating system activities  
- Security Logs  
  - Detect and investigate incidents  
- Application Logs  
  - Activity related to an application  
- Audit Logs  
  - System changes and user events  
- Network Logs  
  - Outgoing and incoming traffic  
- Access Logs  
  - Access to different resources
