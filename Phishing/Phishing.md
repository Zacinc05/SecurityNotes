Social engineering

- Psychological manipulation of people into performing or divulging information by exploiting weaknesses in human nature

Spear-Phishing

- Targeted attack

Anchor text

- Visible, clickable text in an HTML hyperlink

Droppers

- Software that phishing victims tend to be tricked into downloading/running  
- Not usually malicious themselves, tend to pass antivirus checks

Macros

- Embed malicious Visual Basic for Applications (VBA) code within legitimate Office documents

Browser Exploits

- Uncommon  
- Vulnerability against a browser itself which allows the attacker to run remote commands on the victim's computer  
-  ex) CVE-2021-40444, which is a vulnerability found in Microsoft systems that allowed the execution of code just from visiting a website

Choosing a Domain

- Expired Domains  
- Typosquatting  
  - Misspelling  
  - Additional Period  
  - Switching numbers for letters  
  - Phrasing  
  - Additional Word  
- TLD Alternatives  
- Use the same name but with a different TLD (uk/us)  
- IDN Homograph Attack/Script Spoofing  
  - IDN (internationalized domain name) was implemented to support language-specific script or alphabet from other languages  
  - IDN implementation is that different letters from different languages can actually appear identical
