Snort

- Open-source IDS solution  
- Signature-based and anomaly-based detection  
  - Defined in the rule files of the Snort tool  
  - Configure Snort to use custom rules

Snort Modes

- Packet sniffer mode  
  - Read/display network packets without performing analysis  
  - Helpful in network monitoring and troubleshooting  
- Packet logging mode  
  - Detection on the network traffic in real-time  
  - Display detections as alerts for sec admins  
  - Log network traffic as PCAP file  
- NIDS mode  
  - Monitors network traffic in real-time  
  - Applies its rule files to identify a match to stored signature attack patterns
